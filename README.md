# Fitness Tracker App

This repository contains a Flutter-based Fitness Tracker application developed for the CodeAlpha App Development Internship.

## Features

* Add daily fitness activities
* Track exercise duration (minutes)
* Track calories burned
* Display total calories burned
* Simple and user-friendly interface

## Technology Used

* Flutter
* Dart

## How It Works

Users can enter their exercise, duration, and calories burned. The app stores this data and displays all entries along with the total calories burned.
